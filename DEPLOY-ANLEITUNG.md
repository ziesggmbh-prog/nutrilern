# 🚀 Deployment-Anleitung für deine Nutrition Learning Platform

## So findest du den Deploy-Button:

### Option 1: Hauptmenü (empfohlen)
1. **Schaue oben links** in der Replit-Oberfläche
2. **Klicke auf "Deploy"** (sollte neben "Run" stehen)
3. **Wähle "Autoscale"** für bessere Performance
4. **Klicke "Deploy"**

### Option 2: Über das Menü
1. **Klicke auf die drei Striche** ☰ (Hamburger-Menü)
2. **Wähle "Deploy"**
3. **Folge den Anweisungen**

## Was passiert beim Deployment:

✅ **Automatischer Build-Prozess:**
- `npm run build` wird ausgeführt
- Videos werden automatisch kopiert
- Server wird gebaut und optimiert

✅ **Deine Videos sind bereit:**

✅ **Nach dem Deployment:**
- Du bekommst eine URL für deine App
- Videos werden sofort abspielbar sein
- Debug-Route verfügbar unter: `deine-url/debug/video-test`

## Falls Probleme auftreten:

1. **Teste die Debug-Route:** `[deine-deployment-url]/debug/video-test`
2. **Schaue in die Browser-Konsole** (F12 → Console)
3. **Kontaktiere mich** mit der Fehlermeldung
